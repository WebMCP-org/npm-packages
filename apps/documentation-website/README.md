# WebMCP Documentation

Documentation site for [WebMCP](https://github.com/WebMCP-org/npm-packages),
the W3C standard for making websites AI-accessible through
`document.modelContext`.

Built with [Mintlify](https://mintlify.com). Live at [docs.mcp-b.ai](https://docs.mcp-b.ai).

## Development

```bash
npx mintlify dev
```

Preview at http://localhost:3000.
