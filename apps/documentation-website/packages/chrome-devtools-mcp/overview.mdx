`@mcp-b/chrome-devtools-mcp` is retired and no longer maintained from this monorepo.

Use the upstream Chrome DevTools MCP server instead:

- [ChromeDevTools/chrome-devtools-mcp](https://github.com/ChromeDevTools/chrome-devtools-mcp)
- [chrome-devtools-mcp on npm](https://www.npmjs.com/package/chrome-devtools-mcp)

```bash "Install from npm"
npx -y chrome-devtools-mcp@latest
```

The forked WebMCP changes that used to live in `@mcp-b/chrome-devtools-mcp` have been upstreamed. Treat the upstream package as the source of truth for supported CLI flags, tool behavior, release notes, and Chrome compatibility.
