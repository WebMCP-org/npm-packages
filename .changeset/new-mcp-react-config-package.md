Initial release of @mcp-b/mcp-react-config package

New package for exploring and managing MCP configuration files across multiple platforms:

- MCPConfigExplorer component with recursive file system exploration
- ConfigDiffViewer component for side-by-side diff preview
- ConfigFileList component for displaying detected configurations
- Support for 8 platforms (Claude Desktop, Cursor, VSCode, Continue, Cline, Windsurf, Codex, Claude Code CLI)
- Utilities for file system exploration with parent directory tracking
- Configuration generation and merging for JSON, YAML, and TOML formats
- Comprehensive CSS styling with dark mode support
- Full TypeScript types and documentation
